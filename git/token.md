## token使用方式

首先，生成token，然后执行命令：git remote set-url origin  https://<your_token>@github.com/`<USERNAME>`/`<REPO>`.git

示例：`git remote set-url origin https://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@github.com/luo-city/tele-notes.git`
